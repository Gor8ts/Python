Алгоритм керування автомобілем
Рух по прямій
    Стежити за рухом автомобіля по дорозі та уникати перешкод
    Дотримуватись швидкісного режиму та правил дорожнього руху
Рух при повороті
    Увімкнути світловий сигнал повороту лівою рукою
    Завчасно перестроїтися у полосу потрібну для здійснення повороту
        Обома руками повернути кермо в праву сторону для займання правої крайньої смуги
        Або обома руками повернути кермо в ліву сторону для займання лівої крайньої смуги
    Сигналізувати про намір повернути в той чи інший бік переключаючи сигнал повороту лівою рукою
    Зменшити швидкість або зупинитись для того щоб преконатись що не заважаємо руху інших авто
        Для повороту направо
            виїхати на середину перехрестя 
            повернути кермо вправо для руху на право 
            По завершенню повороту вирівняти кермо
        Для повороту на ліво
            виїхати на середину перехрестя
            Дочекатись доки проїдуть всі зустрічні авто
            Повернути кермо вліву сторону
            почати поворот
            по закінченню повороту вирівняти кермо
